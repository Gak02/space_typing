# アプリ企画書：『月面転職クエスト 〜未知の宇宙職業で月を目指せ〜』

## 1. 企画の背景・目的
* **背景:** 宇宙産業は急速に拡大しているが、一般のビジネスパーソンからは「一部の天才エンジニアだけのもの」という心理的ハードルが高い。
* **目的:** 宇宙業界に実在する「ニッチで特殊な職業」とその「泥臭い苦労」をタイピングゲーム化することで、心理的ハードルを下げ、宇宙ビジネスへの興味喚起と認知拡大を図る。
* **ビジネス的価値:** 遊び終わったユーザーを、実際の宇宙スタートアップの採用ページや業界イベントへ送客する、ユニークなHRファネル（採用候補者の入り口）として機能させる。

## 2. ターゲット層
* 20〜40代のビジネスパーソン。
* 日々の業務（調整、稟議、クレーム対応など）に少しお疲れ気味で、非日常感や新しい業界の話題に飢えている層。

## 3. コアコンセプト・ゲームサイクル
RPGのような進行スタイルを取り入れ、プレイヤーが次々と「宇宙特有の職業」にジョブチェンジしながら月を目指すストーリー体験を提供する。

1. **プレイ:** 降ってくる「専門用語」をタイピング。時折、サラリーマンの共感を誘う理不尽な「お邪魔ワード（ニッチな苦労）」が割り込み、これを素早く打って撃退する。
2. **クリア・診断:** ステージをクリアするごとに、タイピングの速度と正確性から「宇宙ビジネス適性パラメータ（正確性、スピード、ストレス耐性など）」が蓄積される。
3. **シェア:** 全ステージクリア後、「あなたの月面での最終役職は【軌道上トラフィックマネージャー】です」といった診断結果の画像が生成され、SNSなどでシェアできる。

## 4. ステージ構成と出題ワード案

### Stage 1：宇宙法務・ルールメイカー
* **ミッション:** 月面資源の所有権を巡る国際法務を切り抜けろ！
* **専門用語:** アルテミス合意 / 損害責任条約 / 宇宙空間平和利用委員会
* **お邪魔ワード:** 「他国からの理不尽なクレーム」「解釈違いによる徹夜の条約翻訳」

### Stage 2：軌道上トラフィックマネージャー
* **ミッション:** 渋滞する地球低軌道で、デブリとの衝突を回避せよ！
* **専門用語:** ケスラーシンドローム / 衝突回避マヌーバ / 能動的デブリ除去
* **お邪魔ワード:** 「大富豪の衛星が道を譲らない」「通信遅延によるニアミス」

### Stage 3：惑星保護官（Planetary Protection Officer）
* **ミッション:** 地球の菌から月を守り抜け！
* **専門用語:** バイオバーデン評価 / クリーンルーム検疫 / 後方汚染阻止
* **お邪魔ワード:** 「技術者が素手で機体を触った」「謎の菌を検出」

### Final Stage：月面居住区アーキテクト
* **ミッション:** 過酷な環境に、人類の新たなオフィスを建設せよ！
* **専門用語:** レゴリス3Dプリント / 地下溶岩チューブ / 閉鎖生態系生命維持システム
* **お邪魔ワード:** 「微小重力下でのトイレ故障」「水資源を巡る社内政治」
* **エンディング:** タイピングをクリアすると、京都、神奈川、東京に次ぐ、自分自身の「第4の生活拠点」として月面オフィスが完成する。

## 5. 実装・システム面の拡張アイデア
* **動的テキスト生成の組み込み:** 出題される「お邪魔ワード」や、最終的な「診断結果テキスト」のバリエーションを増やすため、裏側でLLM（大規模言語モデル）のAPIを連携。プレイヤーごとに少しずつ違うクスッと笑えるテキストを動的に生成し、リプレイ性を高める。
* **データ分析結果のフィードバック:** どの単語でタイピングミスが多かったかを集計し、「あなたは『法務系』の単語入力速度が著しく遅いため、理理尽なクレーム対応に弱い傾向があります」など、データに基づいたユーモアのあるフィードバックを返す。

## 6. 今後の展開（フェーズ分け）
* **フェーズ1（プロトタイプ MVP）:** ブラウザ上で動くシンプルなタイピング機能と、1〜2ステージ分の職業データのみで実装し、まずは知人やSNSでテストプレイを実施。
* **フェーズ2（本開発・連携）:** 全ステージを実装し、結果画面からのSNSシェア機能を強化。実在の宇宙関連企業の採用リンクなどを設置し、送客ツールとしてのテストを開始する。
